## 설치
```
git clone https://github.com/JordanCampos20/Multilingual-Readme-Generator.git
```
```
pip install -r requirements.txt
```

```
python2 및 python3이 있는 경우 requirements.txt를 다운로드할 때 pip3을 사용하세요.
```

## 달리기

```
"main.py" 파일 실행
```
