## 安装
```
git clone https://github.com/JordanCampos20/Multilingual-Readme-Generator.git
```
```
pip install -r requirements.txt
```

```
如果你有python2和python3，下载requirements.txt时使用pip3
```

## 跑步

```
运行“main.py”文件
```
